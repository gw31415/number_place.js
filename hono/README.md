# pg-hono-webapp

## 環境構築

大まかに以下の流れで行います。

1. JS関連ツールのインストール
2. 依存パッケージのインストール

### 1. JS関連ツールのインストール

[`.tool-versions`](./.tool-versions) に書かれたツールを記載されたバージョンでインストールしてください。

> [!TIP]
> [`asdf`](https://asdf-vm.com/) や [`mise`](https://mise.jdx.dev/) を使えば自動でバージョンが合ったものをインストールしてくれます。

### 2. 依存パッケージのインストール

```sh
pnpm i
```
